# Layer 4 - Research Applications

Layer 4 contains the applications that researchers use to move a route through the Research OS lifecycle. These are workflow owners, not loose scripts.

Review focus:

- idea intake and mechanism framing
- data and panel contracts
- feature, factor, state, and label planning
- discovery, review, freeze, confirmation, and closure
- application ownership before lower-layer execution

Representative nodes:

- `idea-intake/`
- `mechanism-memo/`
- `data-panel-contract/`
- `source-label-stage1-trainable/`
- `feature-factor-foundry/`
- `discovery-confirmation/`
- `economic-replay/`

- `apps/`
