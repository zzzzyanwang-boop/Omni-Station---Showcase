# Evidence Envelope

Evidence envelopes package diagnostics, confirmatory results, blocked claims, and artifact references.

Reviewable responsibilities:

- diagnostic versus confirmatory claim separation with explicit stage tags
- evidence reference binding to artifact manifests, gate decisions, and source lineage
- claim scope for universe, horizon, fold policy, model version, replay assumption, and runtime posture
- blocked and deferred claims with required evidence to unblock them
- reviewer-facing summary that can be read without opening raw data, model outputs, or runtime logs
