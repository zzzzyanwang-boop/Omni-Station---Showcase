# Gates

Gates decide whether evidence supports a claim.

Gate decisions should be explicit:

- admitted claim with the exact evidence and scope that support it
- blocked claim with missing, stale, contradictory, or insufficient evidence named
- missing evidence for source lineage, OOF/CPCV, leakage, replay, risk, or runtime eligibility
- scope limitation that prevents overclaiming across universe, horizon, sample, model, or execution assumptions
- downstream eligibility for freeze, confirmation, replay, inference review, closure, or retirement
- reviewer-facing rationale and machine-readable reason code
