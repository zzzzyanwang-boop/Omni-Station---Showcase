# Review Decisions

Review decisions bind evidence results back to governance state.

Reviewable responsibilities:

- freeze decision that locks eligible artifacts, contracts, and confirmation scope
- stop or continue decision with admitted evidence, blocked evidence, and reviewer rationale
- waiver request tied to a named invariant, scope limitation, and expiry condition
- retirement decision that prevents failed or stale routes from re-entering as fresh candidates
- closure case summary with memory entry and downstream eligibility state
