## Current System Shape

The retained paths show a real progression:

```text
source contract
  -> source/label panel materialization
  -> source-backed Stage1 trainable join
  -> formal OOF run-spec gate
  -> full-manifest OOF executor
```

The first two files are artifact-production boundaries. The third file is a formal launch gate. None of them publishes model performance or strategy claims in this showcase.
